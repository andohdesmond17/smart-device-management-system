# smart-device-management-system
# Introduction
The Smart Device Management System is a Python project that demonstrates the principles of Object-Oriented Programming(OOP). The System manages different smart devices used in a smart home. Although every device has common information such as a device ID, device name, and power status, each device also has its own unique features.
The project applies the four main OOP concepts:
-Encapsulation
-Inheritance
-Abstraction
-Polymorphism

# Project Objectives
The project allows users to:
-Turn devices on and off
-Adjust the brightness of a smart light
-Set the temperature of a smart thermostat
-Lock and unlock a smart door lock
-Display information about each device
